## 悦读系统接口定义

### 接口定义
```js
  code： 1表示请求成功, !1表示请求失败
  msg:   表示请求信息，无论请求成功与失败都应该返回
  data:  返回请求的数据
  
  token: 统一放在get请求上
```
### 用户相关
#### 登陆接口
```js
@param username
@param password
@url post  /user/login
@return {
	code :1,
	data: {
    token: 132131321,
    auths: ['admin', 'staff']
	},
	msg： '登陆成功'
}
```

#### 注册接口
